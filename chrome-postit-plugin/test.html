Postit Test Page

This page tests the postit functionality in standalone mode (no extension).

Click the yellow + button (bottom-right) to create a new postit.

Features to test:

  • Create new note (random color, position, rotation)
  • Edit note content (click edit icon on hover)
  • Drag note to reposition
  • Minimize note (goes to tray bottom-left)
  • Restore from tray (click mini note)
  • Delete note
  • Persistence (refresh page, notes should remain)
